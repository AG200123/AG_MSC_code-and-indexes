# AG_MSC_code-and-indexes
This repo contains cleaned up code and image indexes to reproduce the main experiments from "Agricultural Backbone Models for Weed Detection: Evaluating Domain-Specific Transfer Learning".

The indexes for both the Leek and Thistle datasets can be found in the corresponding run scripts.

**The corresponding references are:**

Leek - Lac, L., Keresztes, B., Louargant, M., Donias, M., & Da Costa, J.-P. (2022). An annotated image dataset of vegetable crops at an early stage of growth for proximal sensing applications. Data in Brief, 42, 108035. https://doi.org/10.1016/j.dib.2022.108035

Sorghum - Michael, J., & Thenmozhi, M. (2023). SorghumWeedDataset_Segmentation [Dataset]. Mendeley Data, V1. https://doi.org/10.17632/y9bmtf4xmr.1

Thistle - Coleman, G. (2024). Danish thistles [Dataset]. Weed-AI Sydney. https://weed-ai.sydney.edu.au/datasets/9646f5bf-9cd1-4988-accc-e1663774b046

Rumex - Roboflow 100. (2023). Grass weeds dataset [Dataset]. Roboflow Universe. https://universe.roboflow.com/roboflow-100/grass-weeds


**The pretraining dataset can be found at: **
